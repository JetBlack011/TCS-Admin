theCoderSchool Admin

A simple tool which you can use to block websites and interact with sepcific clients which have the administrative chrome extension installed.

Add a Block

Adding a website to the Blocklist stores it and prevents all connected clients from loading that page. You can also add a specific functionality to how a given client reacts to a request to the blocked page.

View details »

List all Block

All contents of the Blocklist are visible in one location. Moreover, removing a website from the Blocklist is incredibly simple, and can be done with the push of a button. The change is made instantly, and all connected clients must merely refresh the change in order to no longer be affected by the block.

View details »

View Clients

This tool also allows an admin to view information about all connected clients, and issue specific commands to an individual machine.
